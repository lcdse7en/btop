#### 1.btop Install
1. Install dependencies (example for Ubuntu 21.04 Hirsute)
`sudo apt install coreutils sed git build-essential gcc-11 g++-11`

2. Clone repository
`git clone https://github.com/aristocratos/btop.git`
`cd btop`

3. Compile
`make`

4. Install
`sudo make install`
